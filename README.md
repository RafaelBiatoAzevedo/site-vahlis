🏗️ Vahlis Incorporadora - Landing Page
Esta é uma landing page de alto padrão desenvolvida para a Vahlis Incorporadora, focada em consolidar a presença digital de uma marca com quase um século de tradição no mercado imobiliário. O projeto destaca o legado histórico e a inteligência estratégica na prospecção de terrenos e incorporação.

🚀 Sobre o Projeto
A Vahlis Incorporadora (desde 1933) exigia uma interface que transmitisse elegância, solidez e visão de futuro. Esta aplicação foi construída para converter visitantes em investidores através de uma narrativa visual clara sobre:

Legado: Mais de 200 empreendimentos realizados.

Inteligência: Mapeamento estratégico de áreas com alto potencial de valorização.

Excelência: Projetos arquitetônicos alinhados às demandas do mercado premium.

🛠️ Stack Tecnológica
Framework: Next.js 15 (App Router)

Linguagem: TypeScript para tipagem estática e segurança de código.

Estilização: CSS Moderno (com foco em responsividade e performance).

Fontes: Otimização via next/font utilizando a família Geist para uma leitura limpa e moderna.

📦 Funcionalidades Implementadas
Arquitetura de Componentes: Estrutura modular para facilitar a manutenção e escalabilidade.

SEO Otimizado: Configuração de meta-tags e estrutura semântica para melhor ranqueamento.

Performance: Carregamento otimizado de imagens e fontes nativo do Next.js.

Responsividade Total: Design adaptável para Mobile, Tablet e Desktop.

🔧 Como Executar o Projeto
Clone o repositório:

Bash
git clone https://github.com/RafaelBiatoAzevedo/site-vahlis.git
Instale as dependências:

Bash
npm install
# ou
yarn install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Acesse no navegador:
Abra http://localhost:3000 para visualizar o resultado.

🌐 Deploy
O projeto está configurado para deploy contínuo na Netlify. Cada push na branch main gera uma nova versão em produção automaticamente.
